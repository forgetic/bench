# Agent entry point

This repo follows Diátaxis so that each document has a clear job and agents can
load only the context relevant to their task.

## Useful docs

- [Documentation index](docs/README.md) explains the documentation layout.
- [Tutorials](docs/tutorials/README.md) are learning-oriented walkthroughs.
- [How-to guides](docs/how-to/README.md) are task-oriented guides, including
  [using the bench Forgejo helper](docs/how-to/use-the-bench-forgejo-helper.md).
- [Reference](docs/reference/README.md) contains fact-oriented details, including
  the [Forgejo fixture reference](docs/reference/forgejo-fixture.md).
- [Explanation](docs/explanation/README.md) contains background and rationale.

# Documentation

This directory follows the Diátaxis documentation structure for the `bench`
workspace.

- [Tutorials](tutorials/README.md) teach by walking through a task.
- [How-to guides](how-to/README.md) solve practical problems, including
  [using the bench Forgejo helper](how-to/use-the-bench-forgejo-helper.md).
- [Reference](reference/README.md) records factual details, including the
  [Forgejo fixture reference](reference/forgejo-fixture.md).
- [Explanation](explanation/README.md) provides background and rationale.
